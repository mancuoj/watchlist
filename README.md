## Usage

> In WSL-Ubuntu20.04 

```sh
git clone https://github.com/mancuoj/watchlist.git
cd watchlist

python3 -m venv env
. env/bin/activate
pip install -r requirements.txt

flask forge
flask admin
flask run
```
